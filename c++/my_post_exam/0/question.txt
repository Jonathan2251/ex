Write left resursive parser with the following BNF (Bakus Naur Form) and output the assembly for the input

S -> StmtList
StmtList -> Stmt ; StmtList | Stmt
Stmt -> ForStmt | AssignStmt
ForStmt -> "for" Var ":=" Int "to" Int "do" AssignStmt
Var -> {a..z}
Int -> {0..9}
AssignStmt -> Var ":=" Expr
Expr -> Var "+" Int

input:

x:=0;
for i:=1 to 20 do
  x:=x+2;

output:

  mov i, 1
  mov x, 0
loop1:
  bgt i, 20
  addi x, 2
  addi i, 1
  jump loop1
  ret
